For both clone strategies: value and broken-pair-distance, the format of filename is "[instance name]_nsga2_nondominated[run id]", which records the solutions in the last generation for each run.
For [run id], the range is 0~29.
For [instance name], the range is 0~90. Specifically, the instance id is as follows:
